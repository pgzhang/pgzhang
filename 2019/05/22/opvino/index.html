Win10 安装OpenVINO出现CMake Error:" "Cannot open file for write System Error:Permission denied

May 22 2019

  Win10 安装完OpenVINO在测试demo 程序时报如下错误：

  原因是CMake权限不够，但在安装时已经选择了’Add CMake to the system PATH for all users’

  自己安装OpenVINO时安装路径选择在了D盘，最后自己的解决方法是卸载OpenVINO,重新安装，选择默认安装路径，最后成功运行demo程序
